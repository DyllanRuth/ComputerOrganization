A Simple Computer Simulator

This computer demonstrates the workings of a simple computer. It has 100 memory locations in a 10 by 10 grid with addresses 0 to 99 numbered from the top left row by row. The computer has a program counter which is stored in register PC and an accumulator bx. You may single step through a program or run it in one sequence. You may also specify the delay speed to observe the actions of the machine. Memory cells that are referenced become green for one instruction, those that are changed become red. This demonstrates some of the aspects of machine and assembly language programming.

This simulator is a Java applet that has been modified, Original example, Dyllan Ruth, Isaiah Hamilton, Abrianna Asante, Brandon Robinson, Darrell Hayes of Prairie University.

The following table shows the instructions that are available for this computer:

Mnemonic Code (1st word) Code (2nd word) Description
CLAIM 160 n Store the contents of bx in memory location n
PLACE 161 n Load bx with the contents of location n
DO_NEXT 44 n Add n to bx (add immediate)
DO_PREV 45 n Subtract n from bx (subtract immediate)
DO_ADD 50 n Add the contents of location n to bx
DO_SUB 51 n Subtract the contents of location n from bx
GOTO 127 n Jump to the instruction in location n if bx > 0
GOTO2 128 n Jump to the instruction in location n if bx = 0
STOP 0 Halt the execution
